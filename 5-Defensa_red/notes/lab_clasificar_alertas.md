# Práctica de laboratorio. Clasificar alertas
# Objetivos
* __Parte 1__: Investigar ID de Snort y CVE.
* __Parte 2__: clasificar alertas en un entorno Windows.
* __Parte 3__: clasificar alertas en un entorno Unix/Linux.

# Trasfondo/Situación
En esta práctica de laboratorio clasificará las alertas de dos entornos diferentes: Windows y Linux. Usted es responsable de determinar si una alerta detectada es un positivo real o un falso positivo. Usted es responsable de clasificar si las alertas son generadas por el SIEM o por realizar escaneos de vulnerabilidades. Habrá dos entornos diferentes a los que puede aplicar las alertas generadas.

# Recursos necesarios
* Acceso a Internet.

# Instrucciones
## Parte 1: Investigar ID de Snort y CVE
Utilice sitios como mitre.org, snort.org, virustotal.com y sitios web de proveedores para buscar información sobre los siguientes ID de Snort (sid) e ID de CVE. Para cada alerta proporcione la siguiente información:
* La lista cruzada entre ID de Snort y números de CVE cuando esté disponible.
* La puntuación CVSS que indica la gravedad de cada uno de los siguientes ID.
* Una breve descripción de la alerta.
