# Trasfondo/Situación
<a href="https://www.owasp.org/" target="_blank">Open Web Application Security Project (OWASP)</a> es una organización sin fines de lucro dedicada a la seguridad de aplicaciones web. OWASP publica una lista de los 10 problemas de seguridad web más importantes a los que se enfrentan las organizaciones cada tres años aproximadamente. El objetivo de este informe es crear conciencia sobre la seguridad de las aplicaciones web y ayudar a las organizaciones a incorporar esta información en sus procesos para minimizar y mitigar los riesgos de seguridad.

OWASP también acepta contribuciones de la comunidad para contenido relacionado con la seguridad. Las <a href="https://owasp.org/www-community/" target="_blank">OWASP Community Pages</a> proporcionan una lista de ataques y vulnerabilidades reportadas por las comunidades.

En este laboratorio, explorarás el OWASP Top 10 de 2021 y algunos de los ataques y vulnerabilidades reportados por los colaboradores de la comunidad.
